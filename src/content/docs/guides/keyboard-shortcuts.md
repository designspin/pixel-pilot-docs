# Keyboard Shortcuts

Master Pixel Pilot with these keyboard shortcuts.

## Drawing Tools

| Shortcut | Tool |
|----------|------|
| `B` | Pencil Tool |
| `E` | Eraser Tool |
| `G` | Fill Tool |
| `I` | Eyedropper |
| `V` | Move Tool |
| `M` | Rectangular Selection |
| `W` | Magic Wand |
| `L` | Line Tool |
| `U` | Rectangle Tool |
| `O` | Ellipse Tool |
| `H` | Horizontal Mirror Pencil |
| `Shift+H` | Vertical Mirror Pencil |

## File Operations

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New Project |
| `Cmd+O` | Open Project |
| `Cmd+S` | Save Project |
| `Cmd+Shift+S` | Save As |
| `Cmd+E` | Export |

## Editing

| Shortcut | Action |
|----------|--------|
| `Cmd+Z` | Undo |
| `Cmd+Shift+Z` | Redo |
| `Cmd+C` | Copy |
| `Cmd+X` | Cut |
| `Cmd+V` | Paste |
| `Delete` | Delete Selection |

## View

| Shortcut | Action |
|----------|--------|
| `Cmd+=` | Zoom In |
| `Cmd+-` | Zoom Out |
| `Cmd+0` | Reset Zoom |
| `Space` | Pan Canvas (hold and drag) |

## Layers

| Shortcut | Action |
|----------|--------|
| `Cmd+Shift+N` | New Layer |
| `Cmd+J` | Duplicate Layer |

## Animation

| Shortcut | Action |
|----------|--------|
| `[` | Previous Frame |
| `]` | Next Frame |
| `Cmd+D` | Duplicate Frame |
| `Space` | Play/Pause Animation |

## Color

| Shortcut | Action |
|----------|--------|
| `X` | Swap Primary/Secondary Color |
| `D` | Reset to Default Colors |

## Transform

| Shortcut | Action |
|----------|--------|
| `Cmd+H` | Flip Horizontal |
| `Cmd+[` | Rotate 90° CCW |
| `Cmd+]` | Rotate 90° CW |
| `Cmd+Shift+R` | Rotate 180° |
