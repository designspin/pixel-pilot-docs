# Getting Started with Pixel Pilot

Welcome to Pixel Pilot! This guide will help you create your first pixel art project.

## Creating a New Project

1. Launch Pixel Pilot
2. Click **File â†’ New Project** (or press `Cmd+N`)
3. Choose your canvas size:
   - Common sizes: 16x16, 32x32, 64x64
   - Or enter a custom size
4. Click **Create**

## The Interface

### Left Toolbar
Contains your drawing tools:
- **Pencil (B)** - Draw pixels
- **Eraser (E)** - Erase pixels
- **Fill (G)** - Fill areas with color
- **Eyedropper (I)** - Pick colors from the canvas
- **Selection tools** - Select and transform pixels

### Bottom Panel
Manage your animation:
- **Layers** - Organize your artwork
- **Frames** - Create frame-by-frame animation
- **Timeline** - Control animation playback

### Right Panel
- **Canvas preview** - See your full sprite
- **Color palette** - Manage colors
- **Transform tools** - Flip and rotate

## Your First Drawing

1. Select the **Pencil tool** (press `B`)
2. Choose a color from the palette
3. Click on the canvas to draw
4. Use `Cmd+Z` to undo mistakes
