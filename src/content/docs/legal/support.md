## Report a Bug

Found a bug? Please report it on our GitHub Issues page:

[Report an Issue on GitHub](https://github.com/designspin/pixel-pilot/issues)

When reporting a bug, please include:
- Your macOS version
- Pixel Pilot version
- Steps to reproduce the issue
- Expected vs. actual behavior
- Screenshots if applicable

## Frequently Asked Questions

### Is Pixel Pilot free?

Pixel Pilot offers both a free version with basic features and a Pro version with advanced capabilities.

### What file formats does Pixel Pilot support?

- Import: PNG, JPG
- Export: PNG, sprite sheets, animated GIF
- Project files: .pixelpilot format

### What are the system requirements?

- macOS 12.0 or later
- Apple Silicon or Intel Mac

### Can I use Pixel Pilot commercially?

Yes! You can use artwork created with Pixel Pilot for any purpose, including commercial projects.
